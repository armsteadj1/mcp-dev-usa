# @mcp-dev-usa/twilio

Twilio MCP server for AI tool use.

## Quick Start

```bash
npx @mcp-dev-usa/twilio
```

## Environment Variables

TWILIO_ACCOUNT_SID, TWILIO_AUTH_TOKEN

## Claude Desktop Config

```json
{
  "mcpServers": {
    "twilio": {
      "command": "npx",
      "args": ["@mcp-dev-usa/twilio"]
    }
  }
}
```
