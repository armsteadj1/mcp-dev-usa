# @mcp-dev-usa/quickbooks

QuickBooks MCP server for AI tool use.

## Quick Start

```bash
npx @mcp-dev-usa/quickbooks
```

## Environment Variables

QB_ACCESS_TOKEN, QB_REALM_ID

## Claude Desktop Config

```json
{
  "mcpServers": {
    "quickbooks": {
      "command": "npx",
      "args": ["@mcp-dev-usa/quickbooks"]
    }
  }
}
```
