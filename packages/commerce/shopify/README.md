# @mcp-dev-usa/shopify

Shopify MCP server for AI tool use.

## Quick Start

```bash
npx @mcp-dev-usa/shopify
```

## Environment Variables

SHOPIFY_ACCESS_TOKEN, SHOPIFY_STORE_DOMAIN

## Claude Desktop Config

```json
{
  "mcpServers": {
    "shopify": {
      "command": "npx",
      "args": ["@mcp-dev-usa/shopify"]
    }
  }
}
```
